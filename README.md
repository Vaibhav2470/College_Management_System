# 🎓 College Management System

A **College Management System** developed using **VB.NET and Windows Forms** to help manage student records, admissions, and administrative tasks efficiently.
This application provides an easy-to-use interface for college administrators to handle student information and academic operations.

---

## 📌 Features

* 👨‍🎓 **Student Admission Management**
* 🗂 **Student Record Management**
* 🧾 **Admin Dashboard**
* 🔍 **Search and Manage Student Details**
* 🖥 **User-Friendly Windows Forms Interface**
* 💾 **Database Integration for Data Storage**

---

## 🛠 Technologies Used

* **Programming Language:** VB.NET
* **Framework:** .NET Framework (Windows Forms)
* **IDE:** Visual Studio
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository

```
git clone https://github.com/Vaibhav2470/College_Management_System.git
```

2. Open the project in **Visual Studio**

3. Open the solution file:

```
College_Management_System.sln
```

4. Build and run the application.

---

## 🚀 How to Use

1. Launch the application.
2. Navigate through the **Admin Dashboard**.
3. Add or manage **student admission details**.
4. View and manage stored student records.
